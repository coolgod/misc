vi mode:

`q` : quite

`w` : save

`?regex` : search

`split` / `sp` : split window vertically

`vsplit` / `vsp` : split window horizontally

command mode:

`dd`: delete current line

`v` : split window horizontally

`CTRL + W` x 2: switch between window

`CTRL + W, =` : resize window to equal

`gg` : go to start
